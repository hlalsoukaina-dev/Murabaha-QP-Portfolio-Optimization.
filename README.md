# Mourabaha Portfolio Optimization

Projet d'analyse et d'optimisation de portefeuille financier pour les produits Mourabaha.

## 📂 Contenu
- `Murabaha_QP_Framework_2019_2025_v2.ipynb` : Notebook contenant l'analyse complète et le modèle d'optimisation.
- `Murabaha_Quadratic_Ready_Data_2019_2025 (1).xlsx` : Données utilisées.
- `requirements.txt` : Bibliothèques nécessaires.

## 🛠 Utilisation
Ce projet utilise Python et l'algorithme SLSQP pour l'optimisation sous contraintes.
